================================================================================
1. PURPOSE OF THIS CONTRACT
================================================================================

This contract defines the terms and conditions under which
the Service Provider agrees to deliver maintenance services
for the delivered system after project closure.

Maintenance services are limited to preserving system stability,
security, and compliance with approved documentation.

[ES] Este contrato existe para mantener el sistema estable,
[ES] no para seguir construyéndolo.

================================================================================
2. CONTRACT SCOPE
================================================================================

Maintenance services included:
- Corrective maintenance (bug fixing)
- Preventive maintenance (updates, patches)
- Adaptive maintenance (environment compatibility)
- Governance enforcement
- Documentation updates when required
- Support as defined in the SLA

Maintenance services excluded:
- New features or enhancements
- UX/UI redesigns
- Architectural refactoring
- Scope expansion
- Third-party integrations not delivered

[ES] Todo lo excluido requiere un nuevo contrato o adenda.
